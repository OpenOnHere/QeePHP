致谢

写到这里，眼前闪出了不少名字。但我知道这远远不够，还有许多为开源项目作出贡献的朋友已经没有联系了。所以我决定不在这里写出任何一个名字，因为来自社区的每一个贡献者都是不该被遗忘的。我也深深的为没有做好贡献者资料收集与整理工作感到愧疚，没有来自你们的支持、鼓励与批评，就没有今天的 QeePHP。

我们创造，我们分享，只因为心中对美好世界的向往和珍惜，只因为我们也需要信仰！
